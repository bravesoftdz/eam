# eam

O EAM (Emparelhamento ao Modelo) é um programa de computador útil para o planejamento e execução de procedimentos de ensino individualizado. Ele foi deselvolvido em um contexto de pesquisa básica analítico comportamental sobre controle de estímulos e permite apresentação de discriminações sucessivas, simultâneas, simples e condicionais na forma de tentativas discretas, além de procedimentos mais básicos na forma de operante livre. É possível apresentar estímulos (figuras, animações, sons e vídeos) como antecedentes e consequentes a respostas do tipo FR, FT, FI, VR, VT, VI, DRL, DRH e componentes derivados. O código do programa é livre (GPLv3).

Este reservatório é reservado às questões técnicas de desenvolvimento (relacionadas ao código fonte). Por favor, para questões, dúvidas e sugestões gerais utilize o sistema de *issues* do reservatório de documentação (https://github.com/eep-lab/eam_docs/).
